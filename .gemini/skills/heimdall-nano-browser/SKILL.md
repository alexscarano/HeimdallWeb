# Heimdall Nano Browser Integration

Esta skill ajuda você a usar o **Nano Banana no navegador** de forma integrada com o desenvolvimento local.

## Workflow de Integração

1. **Geração de Prompt**: Gere prompts baseados no código real (JSX/CSS).
2. **Uso no Browser**: Copie o prompt para o Nano Banana.
3. **Download e Importação**: Saiba exatamente onde salvar e como importar.

## Image Specialist (Capacidades Extra)
- **Refatoração**: Proponha como transformar rascunhos em assets de UI profissionais.
- **Otimização**: Transforme textos simples em prompts ultra-detalhados (Dark Glass, Soft Glow).
