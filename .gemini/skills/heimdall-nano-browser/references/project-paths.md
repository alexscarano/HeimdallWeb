# Heimdall Image Asset Paths

## Nova UI (Next.js 15)
- **Local**: `src/HeimdallWeb.Next/public/assets/`
- **Pastas**: `/scanners/`, `/branding/`, `/hero/`.
- **Import**: `<Image src="/assets/hero/hero-bg.jpg" ... />`

## UI Legada (ASP.NET)
- **Local**: `HeimdallWebOld/wwwroot/images/`
- **Pastas**: `/icons/`, `/illustrations/`.

## Padrão
Use kebab-case: `scanner-ssl-indigo.png`.
Opacidade para Heros: 0.05 a 0.12.
